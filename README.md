# online-pay
 NewebPay API串接
- 線上交易流程(下圖以「即時支付交易流程」為例)



![image](https://github.com/Josephine-M-Li/online-pay/assets/77156174/f6cc8774-809c-447d-8a3c-253dc32f4e1a)



  ①購買    

  ②傳送交易參數

  ③導向線上支付畫面，填寫、傳送信用卡資料

  ④回傳付款結果

  ⑤回傳交易結果

  ⑥若交易成功，出貨
